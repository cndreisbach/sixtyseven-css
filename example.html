SixtySeven CSS a set of basic styles for simple content

SixtySeven is a set of styles that should work well for standard HTML documents, including ones with form elements. The selectors are all tag-based. You will not need to use classes for SixtySeven to work.

Inspiration

SixtySeven is named after Habitat 67, my favorite example of Brutalist architecture.
